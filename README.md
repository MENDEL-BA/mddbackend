```
cd existing_repo
```

## Run application Back-end 
```
Java 17
Maven 3.3.6

cd existing_repo
git checkout develop

install mysql database
Create Database mdd

Run the main class
```
## Run application Front-end
```

Angular >=14
Node >=16
```
```
cd existing_repo
git checkout main
run npm install
run ng s
```
